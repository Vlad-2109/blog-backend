# Blog service on Express wirh MongoDB

**Here you have next possibpossibilities:**
- JWT auth;
- Registration;
- Login;
- Get user info;
- Get all posts;
- Get post by id;
- Update post by id;
- Create post;
- Delete post;


### For running the project

#### `1) config your project to MongoDB`
#### `2) npm install`
#### `2) npm run start:dev`
